## 📝 구현 세부사항

### 세션 관리
- Cookie 기반 세션 (7일 유효)
- HTTP-only 쿠키 (보안)
- 만료 시간 자동 확인

### 보안
- 비밀번호 bcrypt 해시 (10 rounds)
- 이메일 형식 검증
- 비밀번호 최소 길이 (8자)

### 사용자 플로우
1. 회원가입 → 세션 생성 → 프로젝트 목록
2. 로그인 → 세션 생성 → 프로젝트 목록
3. 프로젝트 상세 → 세션 확인 → 인증되지 않으면 로그인 페이지
