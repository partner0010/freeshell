아래 버튼을 눌러 도메인에 실제로 요청을 보내보고, 결과에 따라 수정해야 할 내용을 안내합니다.

대기 중

테스트 대상

  • Vercel 기본 도메인: https://freeshell.vercel.app
  • 커스텀 도메인: https://freeshell.co.kr

결과

아직 테스트하지 않았습니다.

무엇을 고쳐야 하나요?

테스트 후 안내가 여기에 표시됩니다.

Cloudflare Tunnel / DNS 체크리스트

  • 터널 프로세스 실행 중인지 확인: cloudflared tunnel run <tunnel-name>
  • Cloudflare 대시보드 → Zero Trust / Tunnels: 터널 상태가 Active인지 확인
  • DNS: freeshell.co.kr이 Vercel 배포 URL로 CNAME 설정되어 있는지 확인
  • 터널 재시작 후 2~3분 뒤 커스텀 도메인 재접속

ESLint 경고(이미지 alt, useEffect deps 등)는 빌드 실패와 무관하니 우선 무시해도 됩니다.
