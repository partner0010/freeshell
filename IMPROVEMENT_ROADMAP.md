# 원격 솔루션 개선 로드맵

## 전세계 1% 수준의 원격 지원 솔루션 구현

### 참고 솔루션
- TeamViewer: 가장 널리 사용되는 원격 데스크톱 솔루션
- AnyDesk: 고성능 저지연 원격 제어
- Chrome Remote Desktop: 브라우저 기반 무료 솔루션
- Microsoft Remote Desktop: 엔터프라이즈급 솔루션
