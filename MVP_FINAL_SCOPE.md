# MVP 최종 범위 정의

## 핵심 원칙

### ✅ 포함 기준
1. **수익 모델 포함**: 즉시 수익화 가능
2. **사용자 가치**: 실제 문제 해결
3. **검증 가능**: 사용자 반응 측정 가능
4. **기술적 실현 가능**: 안정적 구현

### ❌ 제외 기준
1. **수익 없는 기능**: 마켓/기업 기능 제외
2. **과도한 기능**: 복잡도 높은 기능
3. **검증 불가능**: 측정 어려운 기능
4. **연구 단계**: 불안정한 기술

## MVP 포함 기능

### 1. 숏폼 생성 (수익: 크레딧) ✅

**기능**:
- 15-60초 숏폼 생성
- 프롬프트 → Script → Scene → Video
- 음성 + 자막
- MP4 결과물

**수익 모델**:
- 크레딧 기반 (텍스트: 1, 이미지: 5, 숏폼: 50)
- 플랜: 무료(100/일) → 스타터($9.99) → 프로($29.99)

**구현 상태**: ✅ 완료
- `orchestrator/shortform/`: 전체 구현 완료

### 2. 사진 → 모션 (수익: 프리미엄) ✅

**기능**:
- 이미지에 모션 적용
- 표정, 눈, 호흡, 고개 움직임
- 프롬프트 기반 선택
- FFmpeg 렌더링

**수익 모델**:
- 프리미엄 기능 (+$19.99/월)
- 또는 크레딧 소비 (모션: 20 크레딧)

**구현 상태**: ✅ 완료
- `orchestrator/motion/`: 전체 구현 완료

### 3. 전문가 도움 (수익: 중개 수수료) ✅

**기능**:
- 전문가 매칭
- 실시간 채팅
- 상태 확인 (기본)
- 리뷰 시스템

**수익 모델**:
- 서비스 수수료: 15-30%
- 전문가 구독: $9.99/월

**구현 상태**: ✅ 완료
- `orchestrator/expert/`: 전체 구현 완료

### 4. Ethics Guard (필수) ✅

**기능**:
- 위험 평가
- 동의 확인
- 차단 플로우
- 사용자 고지

**수익 모델**: 없음 (필수 기능)

**구현 상태**: ✅ 완료
- `orchestrator/ethics/`: 전체 구현 완료

## MVP 제외 기능

### ❌ 교육/전자책 마켓
- **이유**: 수익 구조 복잡, Phase 2
- **대안**: 전문가 프로필에 링크

### ❌ 기업용 원격관리
- **이유**: 인프라 필요, Phase 2
- **대안**: 개인용 상태 확인만

### ❌ 고급 AI 기능
- **이유**: 비용 높음, MVP 불필요
- **대안**: 기본 AI + Rule Fallback

### ❌ 3D 콘텐츠
- **이유**: 복잡도 높음, Phase 2
- **대안**: 2D 이미지 기반

- 재사용률: 30% 이상
