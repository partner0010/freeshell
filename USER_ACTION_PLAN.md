# 📋 사용자 즉시 실행 계획

## ✅ 지금 당장 사용 가능한 기능

### 1. AI 코드 제안 (개선됨) ⚡
**위치**: 에디터 → "AI 분석" 버튼 (Sparkles 아이콘)
**기능**:
- ✅ System Prompt 적용 (코드 편집 전용)
- ✅ Diff 형식 제안 강제
- ✅ 무료 AI 우선 사용 (Groq > Ollama > Together > OpenRouter)
- ✅ 코드를 통째로 덮어쓰지 않음

**사용 방법**:
1. `/editor` 페이지 접속
2. 코드 작성 (HTML/CSS/JS)
3. 상단 "AI 분석" 버튼 클릭
4. Diff 형식 제안 확인
5. 제안 적용/거부

### 2. AI 튜터 패널 (새로 추가) 🆕
**위치**: 에디터 상단 → "AI 튜터" 버튼
**기능**:
- ✅ 코드 설명 요청
- ✅ 실시간 질문/답변
- ✅ 빠른 프롬프트 버튼
- ✅ 무료 AI 사용 (Groq, Ollama 등)

**사용 방법**:
1. 에디터 상단 "AI 튜터" 버튼 클릭
2. 오른쪽에 AI 패널 표시
3. 코드에 대해 질문하거나 빠른 프롬프트 사용
4. AI가 코드를 설명하고 개선 제안

### 3. 기존 기능들 (그대로 사용 가능)
- ✅ **프로젝트 리뷰**: 전체 프로젝트 분석
- ✅ **AI 추천**: 실시간 코드 추천
- ✅ **자동 저장**: 2초마다 자동 저장
- ✅ **버전 히스토리**: 이전 버전 복구
- ✅ **컴포넌트 라이브러리**: UI 컴포넌트 추가
- ✅ **미리보기**: 실시간 미리보기
