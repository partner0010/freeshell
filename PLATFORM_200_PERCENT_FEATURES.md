## 📝 새로운 API 엔드포인트

### 분석 API (`/api/analytics`)
- `GET /overview` - 플랫폼 전체 개요
- `GET /user-growth?days=30` - 사용자 성장 추이
- `GET /video/{video_id}/performance` - 비디오 성과
- `GET /top-videos?limit=10` - 인기 비디오
- `GET /user/{user_id}/engagement` - 사용자 참여도

### 추천 API (`/api/recommendation`)
- `GET /feed?limit=20` - 개인화된 피드
- `GET /trending?limit=20&time_window=7` - 트렌딩
- `GET /similar/{video_id}?limit=10` - 유사 비디오

### 알림 API (`/api/notifications`)
- `GET /?limit=20&unread_only=false` - 알림 조회
- `POST /mark-read/{notification_id}` - 읽음 처리

### SNS API 강화 (`/api/sns`)
- `POST /video/{video_id}/like` - 좋아요
- `DELETE /video/{video_id}/like` - 좋아요 취소
- `POST /video/{video_id}/comment` - 댓글 추가
- `GET /video/{video_id}/comments` - 댓글 조회
