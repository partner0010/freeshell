# 디자인 시스템 재구성 및 우선순위

## 🎯 핵심 원칙
1. **"Most Simple and Most Sophisticated"** - 가장 단순하면서도 정교하게
2. **"Bright and Global"** - 밝고 글로벌한 디자인
3. **"Practical and Usable"** - 실용적이고 사용 가능한 기능
4. **"실제 구동 가능"** - 실제로 작동하는 기능만

## 📊 현재 상태 분석

### ✅ 잘 되어 있는 부분
- 기본 레이아웃 구조
- 색상 시스템 (Blue-Purple-Pink 그라데이션)
- 템플릿 시스템
- AI 해시태그 생성기

### ⚠️ 개선이 필요한 부분
1. **홈페이지 구성이 복잡함**
   - 여러 섹션이 중복되거나 비효율적
   - 사용자 플로우가 명확하지 않음

2. **디자인 일관성**
   - 일부 컴포넌트가 최신 디자인 원칙을 따르지 않음
   - 간격, 타이포그래피 통일 필요

3. **사용되지 않는 기능들**
   - 커뮤니티 관련 페이지들 (사용자 요청으로 제거 예정)
   - 일부 대시보드 기능들

## 🎨 재구성된 디자인 시스템

### 1. 색상 팔레트
```css
Primary: Blue (#6366f1) - 신뢰, 전문성
Secondary: Purple (#8b5cf6) - 창의성, 혁신
Accent: Pink (#ec4899) - 친근함, 활기
Background: White → Light Gray → White (그라데이션)
Text: Gray-900 (주요), Gray-600 (보조)
```

### 2. 타이포그래피
```css
Headings: Bold, Clear Hierarchy
Body: Regular, Readable (16px 기본)
Labels: Medium, Small (14px)
```

### 3. 간격 시스템
```css
Spacing: 4px 단위 (4, 8, 12, 16, 24, 32, 48, 64, 96)
Container: max-w-7xl (1280px)
Section Padding: py-16 (64px)
```

### 4. 컴포넌트 스타일
- **카드**: White background, Subtle shadow, Rounded-xl
- **버튼**: Gradient 또는 Solid, Rounded-lg, Clear states
- **입력 필드**: Clean border, Focus ring, Rounded-xl

## 📐 재구성된 페이지 구조

### 메인 홈페이지 (`/`)
```
1. Hero Section (검색 기능)
   - 간결한 헤드라인
   - AI 검색창
   - "완전 무료" 배지

2. 핵심 기능 (3개 카드)
   - AI 콘텐츠 제작 (→ 프로젝트 생성)
   - 해시태그 생성기
   - 템플릿 라이브러리

3. CTA Section
   - "무료로 시작하기" 버튼
   - 간단한 가격 안내
```

### 프로젝트 관리 (`/projects`)
```
- 프로젝트 목록
- 새 프로젝트 생성 버튼
- 각 프로젝트별 진행 상태 표시
```

### 프로젝트 상세 (`/projects/[id]`)
```
- 프로젝트 정보 (제목, 목적, 타겟 등)
- 5단계 콘텐츠 제작 UI
  - 단계별 진행 바
  - 각 단계 결과 미리보기
  - 재생성 버튼
  - 다음 단계로 진행 버튼
- 템플릿 적용 기능
```

### 템플릿 라이브러리 (`/templates`)
```
- 검색 및 필터
- 템플릿 그리드
- 템플릿 상세 모달
- "템플릿 사용하기" 버튼
```

## 🔄 우선순위별 구현 계획

### Phase 1: 핵심 재구성 (최우선) ⭐⭐⭐
**목표**: 사용자 플로우를 명확히 하고, 가장 중요한 기능부터 완성

1. **홈페이지 재구성** (1일)
   - Hero 섹션 간소화
   - 핵심 기능 3개 카드로 정리
   - 불필요한 섹션 제거

2. **프로젝트 상세 화면** (2일) ⭐
   - 5단계 콘텐츠 제작 UI
   - 단계별 재생성 버튼
   - 진행 상태 표시
   - 템플릿 적용 기능

3. **프로젝트 목록 페이지** (1일)
   - 내 프로젝트 목록
   - 새 프로젝트 생성 버튼
   - 프로젝트 카드 UI

### Phase 2: 결제 및 인증 (2-3일) ⭐⭐
4. **인증 시스템** (1일)
   - 간단한 회원가입/로그인
   - 세션 관리

5. **결제 시스템 기본** (1-2일)
   - Stripe 통합 (최소)
   - 가격 페이지
   - 플랜 선택 UI

### Phase 3: 관리자 기능 완성 (2일) ⭐
6. **관리자 기능 유료화 완료**
   - 각 기능에 플랜 제한 적용
   - 업그레이드 프롬프트 통합

### Phase 4: 템플릿 확장 (지속적)
7. **템플릿 확장**
   - 50개 → 100개
   - 실제 사용 가능한 예시 추가

### Phase 5: 추가 기능 (나중에)
8. SNS 자동 포스팅 (API 연동 필요)
9. 숏폼 콘텐츠 생성기 (추가 개발 필요)
