- **자동 실행**: 콘텐츠 생성 후 자동으로 SEO 최적화

## 🚀 완전 자동화 파이프라인

### 업그레이드된 워크플로우

1. **주제 입력** → 사용자가 주제만 입력
2. **대본 생성** → AI가 자동으로 대본 작성
3. **프롬프트 생성** → AI가 이미지 생성을 위한 프롬프트 자동 생성
4. **이미지 생성** → AI가 이미지 자동 생성
5. **음성 생성** → 대본을 음성으로 변환 (TTS)
6. **영상 생성** → 이미지 + 음성 합성
7. **자막 추가** → 자동 생성 및 스타일링 ✅ NEW!
8. **배경음악 추가** → 자동 선택 및 합성 ✅ NEW!
9. **SEO 최적화** → 제목/설명/태그/해시태그 자동 최적화 ✅ NEW!
10. **분석 및 인사이트** → 성과 분석 및 개선 제안 ✅ NEW!
11. **자동 배포** → YouTube 등 플랫폼에 자동 업로드
