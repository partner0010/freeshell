# 참고 자료 및 유사 서비스

## 유사 서비스 URL

### AI 비디오 생성 플랫폼
1. **Pictory** - https://pictory.ai
   - 텍스트를 비디오로 변환하는 AI 플랫폼
   - 자동 자막 생성 및 편집 기능

2. **InVideo** - https://invideo.io
   - AI 기반 비디오 편집 도구
   - 템플릿 기반 콘텐츠 생성

3. **Loomly** - https://loomly.com
   - 소셜 미디어 콘텐츠 관리 플랫폼
   - 자동 업로드 및 스케줄링

4. **Runway ML** - https://runwayml.com
   - AI 기반 비디오 편집 및 생성
   - 고급 비디오 처리 기능

### YouTube 자동화 도구
1. **TubeBuddy** - https://www.tubebuddy.com
   - YouTube 최적화 도구
   - 키워드 분석 및 제안

2. **VidIQ** - https://vidiq.com
   - YouTube 분석 및 최적화
   - 콘텐츠 아이디어 생성

### 디자인 참고
1. **Bityanolja** - https://bityanolja.com
   - 거래소 스타일의 깔끔한 디자인
   - 다크 테마와 모던한 UI

2. **Binance** - https://www.binance.com
   - 거래소 UI/UX 참고
   - 직관적인 네비게이션

3. **Coinbase** - https://www.coinbase.com
   - 깔끔한 디자인 시스템
   - 사용자 친화적 인터페이스
