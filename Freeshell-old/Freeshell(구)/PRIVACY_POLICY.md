# 개인정보 처리방침

## 📋 수집하는 개인정보

### 필수 수집 항목
- 이메일 주소 (회원가입 시)
- 사용자명 (회원가입 시)

### 선택 수집 항목
- 프로필 정보
- 업로드한 콘텐츠

### 자동 수집 항목
- IP 주소 (보안 및 통계용)
- 접속 로그
- 쿠키 정보

---

## 🔒 개인정보 보호 조치

### 1. 암호화
- 비밀번호: 해시 암호화 저장
- 민감 정보: AES-256-GCM 암호화
- 전송: HTTPS (TLS 1.3)

### 2. 접근 제한
- 최소 권한 원칙
- API 키 기반 인증
- Rate Limiting

### 3. 데이터 보관
- 보관 기간: 1년
- 자동 삭제: 1년 경과 후 자동 삭제
- 백업: 암호화된 백업만 저장

---

## 🌍 GDPR 준수

### 사용자 권리
1. **접근 권리**: 자신의 데이터 조회
2. **수정 권리**: 데이터 수정 요청
3. **삭제 권리**: 데이터 삭제 요청 (잊혀질 권리)
4. **이동 권리**: 데이터 내보내기
5. **거부 권리**: 데이터 처리 거부

### 구현
- API 엔드포인트 제공
- 데이터 내보내기 기능
- 데이터 삭제 기능
