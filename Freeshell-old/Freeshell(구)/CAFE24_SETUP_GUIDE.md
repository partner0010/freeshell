# 카페24 서버 신청 완전 가이드

## 🎯 카페24에서 신청해야 할 서비스

### ✅ 정답: **VPS 서비스** (가상 서버)

카페24에는 여러 호스팅 서비스가 있지만, 우리 프로그램을 실행하려면 **VPS (Virtual Private Server)**가 필요합니다.

---

## 📋 카페24 서비스 종류 설명

### 1. 웹호스팅 (❌ 사용 불가)
- **용도**: 간단한 웹사이트 호스팅
- **제한**: Node.js 실행 불가, 백엔드 서버 실행 불가
- **가격**: 월 3,000원 ~
- **결론**: 우리 프로그램에는 **부적합**

### 2. VPS 서비스 (✅ 우리가 필요한 것!)
- **용도**: 가상 서버 (완전한 서버 환경)
- **장점**: Node.js 실행 가능, 백엔드 서버 실행 가능, 루트 권한
- **가격**: 월 9,900원 ~
- **결론**: **이것을 신청해야 합니다!**

### 3. 클라우드 서버 (✅ 사용 가능하지만 비쌈)
- **용도**: 클라우드 기반 서버
- **가격**: 월 20,000원 ~
- **결론**: VPS보다 비싸지만 더 좋은 성능

### 4. 도메인 (선택사항)
- **용도**: 웹사이트 주소 (예: mycontent.com)
- **가격**: 연간 15,000원 ~
- **결론**: 나중에 구매해도 됨

---

## 🚀 카페24 VPS 신청 방법 (단계별)

### 1단계: 카페24 접속 및 회원가입

1. **카페24 접속**: https://www.cafe24.com
2. **회원가입** (이미 있으면 로그인)
3. **마이페이지** 접속

### 2단계: VPS 서비스 찾기

**방법 1: 직접 링크**
- https://www.cafe24.com/vps/ (VPS 서비스 페이지)

**방법 2: 메뉴에서 찾기**
1. 카페24 메인 페이지
2. 상단 메뉴에서 **"호스팅"** 클릭
3. **"VPS"** 또는 **"가상서버"** 클릭

### 3단계: VPS 플랜 선택

#### 추천: **VPS 스타터 플랜** (가장 저렴)

**사양**:
- CPU: 1코어
- RAM: 1GB
- 스토리지: 20GB SSD
- 트래픽: 무제한
- **가격: 월 9,900원**

**더 나은 성능이 필요하면**:
- VPS 베이직: 월 19,900원 (2코어/2GB/40GB)
- VPS 스탠다드: 월 29,900원 (2코어/4GB/60GB)

### 4단계: 주문하기

1. **"주문하기"** 또는 **"신청하기"** 버튼 클릭
2. **운영체제 선택**: 
   - ✅ **Ubuntu 22.04 LTS** (추천)
   - 또는 Ubuntu 20.04 LTS
3. **결제 정보 입력**
4. **결제 완료**

### 5단계: 서버 정보 받기

결제 완료 후 (보통 몇 분 ~ 몇 시간 내):

**이메일로 받게 되는 정보**:
- ✅ **서버 IP 주소** (예: 123.456.789.0)
- ✅ **SSH 접속 정보**
  - 사용자명: `root` 또는 `ubuntu`
  - 초기 비밀번호 (또는 SSH 키)
- ✅ **서버 관리 페이지 접속 정보**

**예시 이메일**:
```
서버 IP: 123.456.789.0
SSH 사용자명: root
SSH 비밀번호: [임시 비밀번호]
포트: 22
```

3. **자동 배포 스크립트 실행**
   ```bash
   cd backend
   ./scripts/quick-deploy.sh [서버IP] root
   ```

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 웹호스팅과 VPS의 차이는?
**A**: 웹호스팅은 간단한 웹사이트용, VPS는 완전한 서버 환경 (우리는 VPS 필요)

### Q2: 무료 체험이 있나요?
**A**: 카페24는 보통 7일 무료 체험 제공 (신청 시 확인)

### Q3: 서버 정보는 언제 받나요?
**A**: 결제 완료 후 몇 분 ~ 몇 시간 내 이메일로 발송

### Q4: 서버를 중지할 수 있나요?
**A**: 네, 언제든지 중지 가능 (미사용 기간은 비용 없음)

### Q5: 다른 업체도 괜찮나요?
**A**: 네, AWS Lightsail, 가비아 등도 가능 (가이드 참고)
