## ⚠️ 서버 임대 시 주의사항

### 1. 숨겨진 비용 확인
- ❌ **초기 설정비**: 일부 업체는 초기 비용 청구
- ❌ **백업 비용**: 자동 백업 서비스 별도 과금
- ❌ **트래픽 초과**: 일부 업체는 트래픽 초과 시 추가 과금
- ❌ **도메인 비용**: 도메인은 별도 구매 필요 (연간 10,000~20,000원)

### 2. 필수 확인 사항
- ✅ **SSD 스토리지**: HDD보다 빠름
- ✅ **무제한 트래픽**: 트래픽 제한 없는지 확인
- ✅ **백업 서비스**: 자동 백업 포함 여부
- ✅ **24/7 지원**: 문제 발생 시 지원 가능 여부
- ✅ **리뷰 확인**: 다른 사용자들의 리뷰 확인

### 3. 보안 설정
- ✅ **방화벽 설정**: 불필요한 포트 차단
- ✅ **SSH 키 인증**: 비밀번호 대신 키 사용
- ✅ **정기 업데이트**: 시스템 자동 업데이트 설정
- ✅ **SSL 인증서**: HTTPS 사용 (Let's Encrypt 무료)

### 4. 성능 고려사항
- ✅ **CPU 코어**: 최소 1코어 (2코어 권장)
- ✅ **RAM**: 최소 1GB (2GB 권장)
- ✅ **스토리지**: 최소 20GB (40GB 권장)
- ✅ **네트워크**: 한국 서버 권장 (속도)

## 💡 비용 절감 팁

1. **무료 체험 기간 활용**
   - 대부분 업체는 7~30일 무료 체험 제공
   - 체험 후 결정

2. **연간 결제 할인**
   - 연간 결제 시 10~20% 할인
   - 장기 사용 계획이 있다면 고려

3. **리소스 모니터링**
   - 사용하지 않는 리소스 제거
   - 필요 시에만 업그레이드

4. **무료 도메인**
   - Freenom 등에서 무료 도메인 사용
   - 또는 서브도메인 사용
