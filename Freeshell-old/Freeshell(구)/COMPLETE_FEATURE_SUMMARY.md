## 🚀 사용 시나리오

### 시나리오 1: 단일 언어 콘텐츠
1. 주제 입력: "AI로 시작하는 부업"
2. 영상 길이: 10분 선택
3. 원클릭 자동화 실행
4. 결과: 한국어 영상 1개 생성
5. 예상 수익: $10

### 시나리오 2: 전세계 수익화
1. 주제 입력: "AI로 시작하는 부업"
2. 영상 길이: 10분 선택
3. 전세계 수익화 실행
4. 모든 지역 선택
5. 결과: 15개 언어로 영상 생성
6. 예상 수익: $150 (15배 증가!)

### 시나리오 3: 대량 생산
1. 주제 입력: "일일 트렌드"
2. 스케줄링 설정: 매일 자동 생성
3. 전세계 수익화 활성화
4. 결과: 매일 15개 언어로 영상 생성
5. 월 예상 수익: $4,500 (30일 × $150)
