# AWS Lightsail 빠른 가이드 (5분)

## 🎯 지금 할 일

### 1. Lightsail 찾기
- 상단 검색창: **"Lightsail"** 입력
- **"Amazon Lightsail"** 클릭

### 2. 인스턴스 만들기
- **"Create instance"** (큰 파란색 버튼) 클릭

### 3. 설정
1. **지역**: **Asia Pacific (Seoul)** 선택
2. **플랫폼**: **Linux/Unix** 선택
3. **이미지**: **Ubuntu 22.04 LTS** 선택
4. **플랜**: **$3.50/월** (가장 왼쪽) 선택
5. **이름**: `all-in-one-content-ai` 입력

### 4. 생성
- **"Create instance"** 클릭
- 2-3분 대기

### 5. IP 확인
- 생성된 인스턴스 클릭
- **Networking** 탭
- **Public IP** 확인

---

## ✅ 완료!

IP 주소를 알려주시면 배포 도와드립니다!
