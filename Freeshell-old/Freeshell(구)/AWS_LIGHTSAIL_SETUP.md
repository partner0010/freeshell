# AWS Lightsail 신청 완전 가이드

## 🎯 AWS Lightsail이 가장 저렴한 이유

- **가격**: $3.50/월 (약 4,500원)
- **첫 달 무료**: 실제 첫 달 $0
- **서울 리전**: 한국에서 빠른 속도
- **안정성**: 글로벌 서비스

---

## 🚀 5분 안에 신청하기

### 1단계: AWS 계정 생성

1. **AWS 접속**: https://aws.amazon.com
2. **"계정 생성"** 클릭
3. **이메일 주소** 입력
4. **비밀번호** 설정
5. **신용카드 등록** (첫 달 무료이므로 실제 과금 안 됨)
6. **전화번호 인증** 완료

**⏱️ 소요 시간**: 약 5분

---

### 2단계: Lightsail 접속

1. **Lightsail 접속**: https://aws.amazon.com/lightsail
2. **"Create instance"** 버튼 클릭

---

### 3단계: 인스턴스 설정

#### 1. 지역 선택
- **Asia Pacific (Seoul)** 선택 ✅
- 한국에서 가장 빠른 속도

#### 2. 플랫폼 선택
- **Linux/Unix** 선택

#### 3. 블루프린트 선택
- **Ubuntu** 선택
- 버전: Ubuntu 22.04 LTS (최신)

#### 4. 플랜 선택
- **$3.50/월** 선택 ✅ (가장 저렴)
- 사양:
  - CPU: 1코어
  - RAM: 512MB
  - 스토리지: 20GB SSD
  - 트래픽: 1TB

#### 5. 인스턴스 이름
- **all-in-one-content-ai** (또는 원하는 이름)

#### 6. SSH 키 설정
- **기본 키** 선택 (자동 생성)
- 또는 새 키 생성

---

### 4단계: 생성 완료

1. **"Create instance"** 클릭
2. **2-3분 대기** (인스턴스 생성 중)
3. **생성 완료!**

---

## 📝 서버 정보 확인

### IP 주소 확인
1. Lightsail 대시보드에서
2. 생성된 인스턴스 클릭
3. **"Connect using SSH"** 버튼 옆에 **IP 주소** 표시

**예시**: `52.78.123.45`

### SSH 접속 정보
- **사용자명**: `ubuntu` (기본)
- **비밀번호**: 없음 (SSH 키 사용)
- **포트**: 22

---

## 🔑 SSH 키 다운로드

### 방법 1: Lightsail 콘솔 사용 (권장)
1. Lightsail 대시보드
2. **"Connect using SSH"** 버튼 클릭
3. 브라우저에서 바로 접속 가능

### 방법 2: SSH 키 다운로드
1. Lightsail 대시보드
2. **"Account"** → **"SSH keys"**
3. 키 다운로드
4. 로컬에서 사용

---

## 🚀 배포하기

### 서버 IP를 받았으면:

```bash
cd backend
chmod +x scripts/quick-deploy.sh
./scripts/quick-deploy.sh [서버IP] ubuntu
```

**예시**:
```bash
./scripts/quick-deploy.sh 52.78.123.45 ubuntu
```

## 💰 비용 관리

### 첫 달 무료
- AWS 신규 고객은 첫 달 무료
- 실제 첫 달: **$0**

### 이후 비용
- 월 $3.50 (약 4,500원)
- 트래픽 1TB 포함
- 초과 시 추가 과금 (거의 발생 안 함)

### 비용 알림 설정
1. AWS 콘솔 → **"Billing"**
2. **"Budgets"** 설정
3. 월 $5 예산 알림 설정

---

## ⚠️ 주의사항

### 1. 신용카드 등록 필수
- 첫 달 무료이지만 카드 등록 필요
- 실제 과금은 첫 달 이후부터

### 2. 인스턴스 중지
- 사용하지 않을 때는 중지 가능
- 중지 시 비용 발생 안 함 (스토리지만 과금)

### 3. 스냅샷 비용
- 스냅샷(백업)은 별도 과금
- 필요 시에만 생성

---

## 🔒 보안 설정

### 1. 방화벽 설정
Lightsail 대시보드에서:
1. 인스턴스 클릭
2. **"Networking"** 탭
3. **"Firewall"** 설정
4. 포트 추가:
   - 22 (SSH)
   - 80 (HTTP)
   - 443 (HTTPS)
   - 3001 (API)

### 2. SSH 키 보안
- SSH 키는 안전하게 보관
- 공유하지 말 것

## 🐛 문제 해결

### 인스턴스 접속 안 됨
```bash
# SSH 키 권한 설정
chmod 400 [키파일].pem

# 접속 시도
ssh -i [키파일].pem ubuntu@[서버IP]
```

### 비용이 예상보다 많음
1. **Billing** 대시보드 확인
2. 어떤 서비스에 비용이 발생했는지 확인
3. 불필요한 서비스 중지
