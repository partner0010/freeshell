## ⚠️ 부분 구현 (외부 서비스 필요)

### 1. 클라우드 저장소 연동
**현재 상태**: localStorage 기반 시뮬레이션
**실제 연동 필요**:
- Google Drive: OAuth 2.0 인증 + Google Drive API
- Dropbox: OAuth 2.0 인증 + Dropbox API
- GitHub: OAuth 2.0 인증 + GitHub API

**영향**: 프로젝트를 클라우드에 저장/로드하는 기능이 실제로 작동하지 않음

### 2. WebSocket 협업
**현재 상태**: 클라이언트 코드만 구현됨
**실제 연동 필요**:
- WebSocket 서버 구축 (Node.js + Socket.io 또는 ws)
- 실시간 동기화 서버

**영향**: 실시간 협업 기능이 작동하지 않음

### 3. 이메일 전송
**현재 상태**: 개발 환경에서만 콘솔 출력
**실제 연동 필요**:
- SendGrid, AWS SES, Nodemailer 등 이메일 서비스 연동

**영향**: 팀 초대, 비밀번호 재설정 이메일이 실제로 전송되지 않음

### 4. 데이터베이스
**현재 상태**: 메모리 기반 (서버 재시작 시 데이터 손실)
**실제 연동 필요**:
- PostgreSQL + Prisma 마이그레이션
- 또는 MongoDB, MySQL 등

**영향**: 서버 재시작 시 모든 데이터 손실

### 5. 관리자 분석 데이터
**현재 상태**: 시뮬레이션 데이터
**실제 연동 필요**:
- 실제 사용자 활동 추적
- 데이터베이스에서 통계 조회

**영향**: 관리자 대시보드의 통계가 실제 데이터가 아님
