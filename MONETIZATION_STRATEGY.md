# 수익화 전략 문서

## 🎯 핵심 목표
**"실제로 돈이 되는 방향 위주"** 수익 엔진 구축

## 💰 수익 모델

### 1. 구독 기반 SaaS (Software as a Service)
- **월간/연간 구독 모델**
- **4단계 플랜**: 무료 → 개인 → 프로 → 엔터프라이즈
- **자동 갱신 및 취소 기능**

### 2. 기능별 유료화 전략

#### A. 콘텐츠 제작 서비스
- **무료 플랜**: 기본 3단계까지만 (PLAN, STRUCTURE, DRAFT)
- **유료 플랜**: 품질 개선(QUALITY) 및 플랫폼 변환(PLATFORM) 포함
- **프리미엄 AI 모델**: Pro 플랜 이상에서 고급 AI 모델 사용
- **템플릿 라이브러리**: 유료 플랜에서 더 많은 템플릿 제공

#### B. 관리자 도구 (B2B 유료 서비스)
기존 관리자 기능을 기업 대상 유료 서비스로 전환:

1. **전자결재 시스템** (Electronic Signature)
   - 무료: ❌ 사용 불가
   - 개인: ✅ 기본 기능 (월 제한)
   - 프로: ✅ 고급 워크플로우, 보안 강화
   - 엔터프라이즈: ✅ 모든 기능, 화이트라벨

2. **시스템 진단** (System Diagnostics)
   - 무료: ❌ 사용 불가
   - 개인: ✅ 월 10회 스캔
   - 프로: ✅ 월 100회 스캔, 자동 보고서
   - 엔터프라이즈: ✅ 무제한, 실시간 모니터링

3. **디버그 도구** (Debug Tools)
   - 무료: ❌ 사용 불가
   - 개인: ❌ 사용 불가
   - 프로: ✅ 코드 분석, 성능 최적화
   - 엔터프라이즈: ✅ 모든 기능, 우선 지원

4. **사이트 검사** (Site Check)
   - 무료: ❌ 사용 불가
   - 개인: ✅ 월 10회 검사
   - 프로: ✅ 월 100회 검사, AI 기반 최적화 제안
   - 엔터프라이즈: ✅ 무제한, 실시간 보안 모니터링

5. **원격 솔루션** (Remote Solution)
   - 무료: ❌ 사용 불가
   - 개인: ✅ 월 5회 세션
   - 프로: ✅ 월 50회 세션, 고급 기능
   - 엔터프라이즈: ✅ 무제한, 전담 지원

## 🎯 타겟 고객층

### 1. 무료 플랜
- **타겟**: 개인 사용자, 학생, 취미 활동자
- **목적**: 서비스 체험, 기본 기능 사용
- **전환 전략**: 기능 제한으로 프리미엄 기능 체험 유도

### 2. 개인 플랜 (₩9,900/월)
- **타겟**: 프리랜서, 개인 창작자, 소규모 블로거
- **목적**: 기본 콘텐츠 제작 + 제한적 관리자 도구
- **핵심 가치**: 접근성, 합리적 가격

### 3. 프로 플랜 (₩29,900/월)
- **타겟**: 소규모 팀, 전문가, 중소기업
- **목적**: 고급 콘텐츠 제작 + 전문 관리자 도구
- **핵심 가치**: 생산성, 전문성

### 4. 엔터프라이즈 플랜 (₩99,000+/월)
- **타겟**: 대기업, 대규모 조직
- **목적**: 모든 기능 + 맞춤형 솔루션
- **핵심 가치**: 확장성, 전담 지원, 화이트라벨

## 🔄 전환 전략

### 무료 → 유료 전환
1. **3단계 이후 결제 유도**: QUALITY 단계 접근 시 업그레이드 안내
2. **플랫폼 변환 전용**: PLATFORM 단계는 유료 전용
3. **기능 제한**: 관리자 도구 접근 시 업그레이드 안내
4. **무료 체험**: 신규 사용자 7일 무료 체험 제공

### 플랜 업그레이드 유도
1. **사용량 기반 제안**: 한도에 근접 시 업그레이드 제안
2. **프리미엄 기능 강조**: 업그레이드 시 추가 혜택 표시
3. **할인 혜택**: 연간 구독 시 할인 제공
