## ⚠️ 부분 구현 (외부 서비스 연동 필요)

### 1. 클라우드 저장소 연동
**현재**: localStorage 기반 시뮬레이션
**필요**: OAuth 2.0 인증 + 각 서비스 API
- Google Drive API
- Dropbox API  
- GitHub API

**영향**: 프로덕션에서는 실제 클라우드 저장이 작동하지 않음 (개발/테스트용으로는 작동)

### 2. WebSocket 실시간 협업
**현재**: 클라이언트 코드만 구현
**필요**: WebSocket 서버 구축
- Node.js + Socket.io 또는 ws
- 실시간 동기화 서버

**영향**: 실시간 협업 기능이 작동하지 않음 (단일 사용자 사용 시 문제 없음)

### 3. 이메일 전송
**현재**: 개발 환경에서 콘솔 출력
**필요**: 이메일 서비스 연동
- SendGrid, AWS SES, Nodemailer 등

**영향**: 팀 초대, 비밀번호 재설정 이메일이 실제로 전송되지 않음

### 4. 데이터베이스
**현재**: 메모리 기반 (서버 재시작 시 데이터 손실)
**필요**: 실제 데이터베이스
- PostgreSQL + Prisma (권장)
- 또는 MongoDB, MySQL

**영향**: 서버 재시작 시 모든 데이터 손실 (개발/테스트용으로는 문제 없음)

### 5. 관리자 분석 데이터
**현재**: 시뮬레이션 데이터
**필요**: 실제 사용자 활동 추적 및 데이터베이스 조회

**영향**: 관리자 대시보드의 통계가 실제 데이터가 아님 (기능 자체는 정상 작동)
