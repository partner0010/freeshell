- **XSS 방어** - HTML 살균, CSP 적용, DOM 보호

- **SQL Injection 방지** - 입력 검증, 패턴 탐지
- **Rate Limiting** - IP/엔드포인트별 제한
- **DDoS 방어** - 자동 공격 탐지, IP 차단
- **보안 헤더** - 15+ 헤더 적용 (HSTS, CSP, COEP 등)
- **암호화** - AES-256-GCM, RSA-4096, ECDSA P-384
- **인증/인가** - RBAC, JWT, 계정 잠금, 2FA 준비
- **모의해킹** - 자동화된 40+ 테스트 케이스
