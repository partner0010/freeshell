## 📚 주요 기능 가이드

### 성능 최적화 사용
```typescript
import { memoized, useDebounceCallback } from '@/lib/performance/component-optimizer';

const OptimizedComponent = memoized(MyComponent);
const debouncedSearch = useDebounceCallback(handleSearch, 300);
```

### 코드 검증
1. "코드검증" 메뉴 클릭
2. 코드 입력 또는 "프로젝트 전체 검증"
3. 오류 및 경고 확인

### 원격 지원
1. "원격지원" 메뉴 클릭
2. "새 세션 생성" 클릭
3. 생성된 코드를 공유
4. 상대방이 코드로 연결
