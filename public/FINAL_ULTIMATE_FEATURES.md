## 🚀 사용 방법

### 멀티모달 AI:
1. 사이드바 → 멀티모달 AI
2. 변환 타입 선택 (텍스트→이미지, 이미지→코드 등)
3. 입력 내용 입력
4. "변환 실행" 버튼 클릭

### 버전 비교:
1. 사이드바 → 버전 비교
2. 두 버전의 코드 입력
3. "버전 비교" 버튼 클릭
4. 변경사항 및 통계 확인

### 전체 검색:
1. 사이드바 → 전체 검색
2. 검색어 입력
3. 결과에서 선택하여 실행
