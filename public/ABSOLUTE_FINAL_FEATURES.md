## 🚀 사용 방법

### AI 프레젠테이션:
1. 사이드바 → 프레젠테이션
2. 주제 입력 및 슬라이드 수 설정
3. "프레젠테이션 생성" 클릭
4. PDF/PPTX/HTML로 내보내기

### 와이어프레임:
1. 사이드바 → 와이어프레임
2. 템플릿 선택 또는 새로 생성
3. 요소 추가 및 편집
4. JSON/PNG/SVG로 내보내기

### 분석 대시보드:
1. 사이드바 → 분석대시보드
2. 시간 범위 선택 (일/주/월/년)
3. 통계 및 차트 확인
4. 데이터 분석

### 실시간 채팅:
1. 사이드바 → 실시간채팅
2. 메시지 입력
3. 실시간 소통
