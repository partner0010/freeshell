## 🚀 사용 방법

### 차트 빌더:
1. 사이드바 → 차트 빌더
2. 차트 타입 선택 (라인, 바, 파이 등)
3. "생성" 버튼 클릭
4. 차트 미리보기 확인 및 설정 내보내기

### 워크플로우 빌더:
1. 사이드바 → 워크플로우
2. 워크플로우 이름 입력 후 생성
3. 노드 추가 및 연결 (시각적 에디터 구현 필요)
4. 워크플로우 실행

### 향상된 커맨드 팔레트:
- **Ctrl+K** (또는 Cmd+K)로 열기
- 키워드로 검색
- 화살표 키로 이동
- Enter로 실행
