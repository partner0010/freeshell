## ⚠️ 발견된 잠재적 문제점

### 1. API 키 의존성
- 일부 기능이 API 키 없이 작동하지 않을 수 있음
- Fallback 메커니즘이 제대로 작동하는지 확인 필요

### 2. WebRTC 원격 솔루션
- 브라우저 호환성 문제 가능
- STUN/TURN 서버 설정 확인 필요

### 3. 학습 데이터 저장
- `.ai-learning` 디렉토리 생성 권한 문제 가능
- 서버 사이드에서만 작동하는지 확인 필요

### 4. Ollama 의존성
- Ollama가 설치되지 않은 환경에서 작동 확인 필요
