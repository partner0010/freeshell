# 전문가 시스템 구조

## 기능 범위

### 1. 원격 지원 (Remote Support)
- **PC 원격 제어**: 사용자 PC에 안전하게 접속하여 문제 해결
- **서버 관리**: 서버 상태 모니터링 및 관리
- **실시간 화면 공유**: 화면 공유를 통한 가이드
- **파일 전송**: 안전한 파일 전송 및 백업

### 2. 상태 확인 (Health Check)
- **시스템 진단**: CPU, 메모리, 디스크 상태
- **네트워크 진단**: 연결 상태, 속도 측정
- **보안 스캔**: 바이러스, 취약점 검사
- **성능 분석**: 병목 지점 식별

### 3. 보안 점검 (Security Audit)
- **취약점 스캔**: 시스템 취약점 검사
- **보안 설정 검토**: 방화벽, 권한 설정
- **로그 분석**: 이상 행위 탐지
- **보안 권장사항**: 개선 방안 제시

### 4. 복구/포렌식 (Recovery/Forensics)
- **데이터 복구**: 삭제된 파일 복구
- **시스템 복구**: 손상된 시스템 복구
- **포렌식 분석**: 보안 사고 분석
- **백업/복원**: 데이터 백업 및 복원

### 5. 전문가 매칭 (Expert Matching)
- **스킬 기반 매칭**: 전문가 스킬과 요청 매칭
- **평점 시스템**: 전문가 평점 및 리뷰
- **가격 제안**: 전문가별 가격 제안
- **실시간 채팅**: 전문가와 실시간 소통

### 6. 마켓플레이스 (Marketplace)
- **전자책 판매**: 기술 가이드, 튜토리얼
- **강의 판매**: 온라인 강의, 워크샵
- **템플릿 판매**: 코드 템플릿, 설정 파일
- **컨설팅 서비스**: 시간당 컨설팅

## 사용자 → 전문가 흐름

```
1. 사용자 요청
   ↓
2. 문제 분류 (자동)
   ↓
3. 전문가 매칭
   ├─ 스킬 매칭
   ├─ 가용성 확인
   └─ 가격 제안
   ↓
4. 전문가 선택
   ↓
5. 서비스 제공
   ├─ 원격 지원
   ├─ 상태 확인
   ├─ 보안 점검
   └─ 복구/포렌식
   ↓
6. 완료 및 결제
   ↓
7. 리뷰 및 평점
```

## 수익 분배 구조

### 서비스 수익 분배
- **플랫폼 수수료**: 20-30%
- **전문가 수익**: 70-80%
- **결제 수수료**: 별도 (2-3%)

### 마켓플레이스 수익 분배
- **전자책/강의**: 플랫폼 30% / 전문가 70%
- **템플릿**: 플랫폼 20% / 전문가 80%
- **컨설팅**: 플랫폼 15% / 전문가 85%

### 구독 모델
- **전문가 프리미엄**: 월 $9.99 (수수료 15%)
- **일반 전문가**: 월 $0 (수수료 30%)

## MVP 포함 여부

### Phase 1 (MVP) - 포함
- ✅ 기본 전문가 매칭
- ✅ 실시간 채팅
- ✅ 간단한 상태 확인
- ✅ 기본 리뷰 시스템

### Phase 1 (MVP) - 제외
- ❌ 원격 PC 제어 (보안 복잡)
- ❌ 서버 관리 (인프라 필요)
- ❌ 포렌식 (전문 도구 필요)
- ❌ 마켓플레이스 (Phase 2)

### Phase 2 - 추가
- 원격 지원 (기본)
- 보안 점검 (자동화)
- 전자책 마켓

### Phase 3 - 추가

- 서버 관리
- 포렌식 도구
