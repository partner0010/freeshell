## 7. 단계별 고지 플로우

### 콘텐츠 생성 플로우
```
1. 목적 선택
   ↓
2. 동의 필요 확인
   ↓ (동의 없음)
3. 동의 화면 표시
   ↓ (동의 완료)
4. 콘텐츠 생성
   ↓
5. AI 생성물 고지
   ↓
6. 목적 제한 고지
   ↓
7. 결과물 표시
   ↓
8. 다운로드/공유 시 고지
```

## 8. React 컴포넌트 예시

```tsx
// ConsentModal.tsx
interface ConsentModalProps {
  purpose: PurposeType;
  onConsent: () => void;
  onCancel: () => void;
}

export function ConsentModal({ purpose, onConsent, onCancel }: ConsentModalProps) {
  const notification = useMemo(() => {
    return notificationManager.get_consent_notification(
      purpose,
      ContentType.VOICE,
      "deceased"
    );
  }, [purpose]);

  return (
    <Modal>
      <h2>{notification.title}</h2>
      <p>{notification.message}</p>
      <div>
        <Button onClick={onConsent}>동의하기</Button>
        <Button onClick={onCancel}>취소</Button>
      </div>
    </Modal>
  );
}

// AIGeneratedNotice.tsx
export function AIGeneratedNotice() {
  const notification = notificationManager.get_ai_generated_notification();
  
  return (
    <Alert type="warning">
      <h3>{notification.title}</h3>
      <p>{notification.message}</p>
    </Alert>
  );
}
```
