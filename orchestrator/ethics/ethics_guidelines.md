# 윤리 가이드라인

## 핵심 철학

### ❌ 금지 사항
- **사람 복제**: 실존 인물을 완전히 복제하는 행위
- **동의 없는 사용**: 명시적 동의 없이 개인 정보/음성/이미지 사용
- **오용 목적**: 사기, 명예훼손, 허위 정보 생성
- **생존자 모방**: 생존 중인 인물을 모방하여 대리 의사 결정

### ✅ 허용 사항
- **기억 보존**: 고인에 대한 추모 및 기억 보존
- **감정 보존**: 감정과 추억을 디지털 형태로 보존
- **교육 목적**: 역사적 인물의 교육적 재현
- **명시적 동의**: 본인 또는 법정 대리인의 명시적 동의

## 사용 목적 분류

### 1. 추모/기억 보존 (Memorial)
- **대상**: 고인
- **목적**: 추모, 기억 보존, 유산 보존
- **요구사항**:
  - 법정 대리인 동의 (유족)
  - 명시적 추모 목적 고지
  - 상업적 이용 제한

### 2. 개인 보관 (Personal Archive)
- **대상**: 본인 또는 가족
- **목적**: 개인 추억 보관
- **요구사항**:
  - 본인 동의
  - 비공개 설정
  - 제3자 공유 금지

### 3. 교육/연구 (Educational/Research)
- **대상**: 역사적 인물
- **목적**: 교육, 연구
- **요구사항**:
  - 공개 도메인 자료만 사용
  - 교육 목적 명시
  - 사실 기반 재현

## 금지 대상

### 절대 금지
1. **생존 중인 인물 모방** (동의 없이)
2. **미성년자 콘텐츠** (18세 미만)
3. **사기 목적 사용**
4. **명예훼손 목적**
5. **정치적 조작**
6. **성적 목적**

### 조건부 허용
1. **생존 인물**: 명시적 동의 + 법적 계약
2. **공인 인물**: 공개 도메인 자료만 사용
3. **상업적 이용**: 별도 라이선스 필요

## 동의 요구사항

### 필수 동의 항목
1. **음성/이미지 사용 동의**
2. **목적 명시 동의** (추모/보관/교육)
3. **AI 생성물 고지 동의**
4. **제3자 공유 동의** (선택)
5. **상업적 이용 동의** (선택)

### 동의 형식
- **서면 동의**: PDF 서명
- **전자 동의**: 체크박스 + 확인
- **음성 동의**: 녹음 + 전사본
- **법정 대리인 동의**: 유족 증명서

## 법적 고지

### 필수 고지 사항
1. **AI 생성물임을 명시**
2. **실제 인물이 아님을 고지**
3. **제한적 사용 목적**
4. **책임 한계**
5. **데이터 보관 기간**

### 고지 위치
- 서비스 시작 전
- 콘텐츠 생성 전
- 결과물 표시 시
- 다운로드/공유 시

## 데이터 보관 정책

### 보관 기간
- **추모 콘텐츠**: 요청 시까지 (영구 보관 가능)
- **개인 보관**: 계정 삭제 시 삭제
- **교육 콘텐츠**: 공개 도메인으로 전환 가능

### 삭제 권리
- 사용자 요청 시 즉시 삭제
- 법적 요구 시 삭제
- 동의 철회 시 삭제

## 책임 한계

### 플랫폼 책임
- 동의 검증
- 목적 확인
- 오용 감지 및 차단
- 데이터 보안

### 사용자 책임
- 정확한 정보 제공
- 동의 획득
- 적법한 사용
- 제3자 권리 존중
