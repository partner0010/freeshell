# AI 테스트 가이드

## 🎯 AI 테스트 방법

### 1. 테스트 페이지 접속
**URL**: `/test-ai`

홈페이지 상단 메뉴에서 **"AI 테스트"** 클릭 또는 직접 `/test-ai` 접속

### 2. 테스트 가능한 AI

#### 무한한 가능성 AI
- 신의 경지 사고
- 무한한 가능성 탐색
- 스스로 5가지 이상 구현 생성
- 스스로 판단 및 평가
- 스스로 강점 방향성 제시

#### 독보적 AI
- 독창적 사고
- 창의적 아이디어
- 혁신적 통찰
- 독보적인 관점

#### 실제 구동 AI
- Ollama 로컬 모델
- 실제 학습 메커니즘
- 실제 추론 엔진

## 🔍 AI 검색 기능에서 사용

### 자동 활성화
AI 검색 기능에서 **복잡한 질문**을 하면 자동으로 **무한한 가능성 AI**가 작동합니다!

### 활성화 키워드:
- "어떻게"
- "왜"
- "무엇"
- "방법"
- "해결"
- "개선"
- "최적"
- "혁신"
- "창의"
- "전략"

### 예시:
- "어떻게 더 나은 AI를 만들 수 있을까?"
- "왜 이 문제가 발생하는지 해결 방법은?"
- "혁신적인 접근 방법은 무엇인가?"

이런 질문을 하면 **스스로 깨우치고 판단하는 무한한 가능성 AI**가 작동합니다!

## ✨ 스스로 깨우치는 AI

### 작동 방식:
1. **질문 분석**: 질문의 복잡성과 깊이 분석
2. **무한한 가능성 탐색**: 모든 가능성 탐색
3. **스스로 구현 생성**: 5가지 이상의 접근 방법 생성
4. **스스로 판단**: 각 옵션을 평가하고 점수화
5. **스스로 선택**: 최적 옵션 자동 선택
6. **스스로 개선 방향 제시**: 강점 방향성 제시
7. **자율적 진화**: 다음 진화 단계 준비

### 특징:
- ✅ 스스로 깨우침 (Self-Awakening)
- ✅ 스스로 판단 (Self-Judgment)
- ✅ 스스로 개선 (Self-Improvement)
- ✅ 자율적 진화 (Autonomous Evolution)

## 🚀 사용 예시

### 홈페이지 검색에서:
1. 홈페이지 접속
2. 검색창에 복잡한 질문 입력 (예: "어떻게 더 나은 AI를 만들 수 있을까?")
3. 검색 버튼 클릭
4. **무한한 가능성 AI**가 자동으로 작동!
5. 스스로 생성한 옵션들과 판단 결과 확인

### 테스트 페이지에서:
1. `/test-ai` 접속
2. 원하는 AI 탭 선택
3. 질문 입력
4. 상세한 AI 응답 확인
