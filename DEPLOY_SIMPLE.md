# 간단한 배포 방법

## 방법 1: Netlify에서 직접 배포 (가장 간단)

1. https://app.netlify.com 접속
2. 프로젝트 선택 (freeshell.co.kr)
3. **"Deploys"** 탭 클릭
4. **"Trigger deploy"** 버튼 클릭
5. **"Deploy site"** 선택

끝!

## 방법 2: 배치 파일 사용

1. Cursor에서 `.github` 폴더 열기
2. `deploy.bat` 파일 더블클릭
3. 모든 질문에 Enter 또는 Y 입력
4. 완료!

## 문제가 있으면

배치 파일이 안 되면 방법 1을 사용하세요.
